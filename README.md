# To-Do List Application

## Description
This is a simple To-Do List application developed using Python Tkinter.

## Features
- Add tasks
- Delete selected tasks
- Clear all tasks

## Requirements
- Python 3.x
- Tkinter (comes with Python)

## How to Run

1. Open terminal.
2. Navigate to project folder.
3. Run:

python todo.py

## Concepts Used
- Tkinter GUI
- Listbox
- Buttons
- Functions
- Event Handling
